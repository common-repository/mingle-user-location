== Installation ==

1. Upload the folder Mingle-User-Location.zip to your Wordpress plugin directory.
2. Activate the plugin from the plugin panel. Make changes to background and text colors in the admin section for Mingle User Locations
3. Navigate to the widget configuration panel and add to desired sidebar location.
4. You can change the number of users it shows in Userloc.php on line 83. Change the 5 to the quanity you want shown. I will add an admin option for this later.
